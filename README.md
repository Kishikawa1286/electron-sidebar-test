# electron-sidebar-test

## 環境構築手順
- リポジトリをクローン
```
git clone https://github.com/Kishikawa1286/electron-sidebar-test.git
```
- パッケージをインストール
```
cd electron-sidebar-test
yarn install
```

## 実行
- ビルドしてelectronで起動
```
yarn electron-start
```

### 補足
アプリとしてビルドするためのディレクトリ構成にはなっていません。
スクリプトを実行してもエラーになります。
